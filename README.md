# Pygame-Modding
This is a full tutorial for modding in pygame that uses a small tutorial I made for this, during this little course you will learn how to modify the tutorial game and create a package that can be added during loading 

#
## Questions you might have
### Is Modding in pygame difficult?
###### No, Modding in pygame is not difficult, as all you are doing is changing the base code to add your own twist.

### Is Modding in pygame easy?
###### No and Yes, Modding in pygame is relativly easy but it will take sometime for you to fully understand what you are doing and what you are changing.
